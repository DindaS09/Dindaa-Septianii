# 🍰 BoluKu - E-Commerce Toko Bolu Premium

## 📖 Deskripsi

**BoluKu** adalah website e-commerce sederhana yang dibuat menggunakan **HTML, CSS, dan JavaScript** tanpa menggunakan database maupun framework. Data pengguna dan keranjang belanja disimpan menggunakan **Local Storage**, sehingga aplikasi dapat dijalankan secara lokal menggunakan Live Server.

Website ini dibuat sebagai project pembelajaran dalam pembuatan aplikasi e-commerce berbasis web.

## 🚀 Cara Menjalankan

1. Download atau clone project ini.

2. Buka folder project menggunakan **Visual Studio Code**.

3. Install extension **Live Server**.

4. Klik kanan pada file **index.html**.

5. Pilih **Open with Live Server**.

6. Website siap digunakan.
